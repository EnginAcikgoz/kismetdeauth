# kismetdeauth
Kismet plugin for deauth all AP

Installation:
```sh
$ git clone https://github.com/EnginAcikgoz/kismetdeauth.git
$ cd kismetdeauth
$ chmod +x kismetdeauth.py
```

Usage:
```sh
$ ./kismetdeauth.py -i INTERFACE
```
